分享一些实用工具和平台
 2019-04-23   
 482 words    1 mins   

这个帖子用来不定期更新网上收集的一些实用工具和平台，运用得好还是可以提高学习或工作效率的。
PS：部分资源国内无法访问，这个问题自己想办法解决。

  • 社区和博客 ：
    StackOverflow ——> 最棒的编程问答社区
    SegmentFault ——> 国内编程问答社区
    掘金 ——> 后起之秀，文章质量很高
    Bits and pieces ——> 不错的技术博客网站（好像和Medium有关系？）
    CSDN ——> 界面很乱，不是很喜欢
    博客园 ——> 优秀文章和大牛聚集地
    FreeCodeCamp ——> FreeCodeCamp中文社区

  • 文件编辑类：
    Online converter ——> 文件格式转换
    PDF To Word ——> pdf 转 word
    Online PDF to EPUB converter ——> pdf 转 epub
    Table Convert Online ——> 表格转换
    Compress images ——> 图片压缩
    Smart Resize ——> 图片修改
    Syntax-highlight-word ——> 可以在 word 中使用代码高亮
    tool.lu ——> 在线编写 markdown 以及 html 和 md 转换

  • 电子书：
    GitHub coding books online reading ——> Github编程相关书籍分享
    itbook.download ——> 编程相关书籍免费下载
    Jiumo Search ——> 目前用过最棒的电子书搜索网站
    Ebookee ——> 编程相关书籍免费下载（外文）

  • 搜索引擎：
    chongbuluo search ——> 集成搜索网站
    DuckDuckGo ——> 搜索引擎

  • 前端相关：
    MDN Web Doc ——> MDN文档
    ECMAScript International ——> ECMAScript 标准
    CSS速查总表 ——> 速查属性
    Caniuse ——> 前端兼容性自查
    SM.MS Picture ——> 免费图床
    Font Awesome ——> 图标库
    Iconfont ——> 图标网站
    free icons ——> 图标网站
    Clippy — CSS clip-path maker ——> 前端clip工具
    fontke ——> 字体网站
    ScreenDump ——> 响应式效果测试
    JavaScript Visualizer ——> 可视化JavaScript执行过程（仅ES6之前）
    Loupe ——> 可视化事件循环 （Philps Roberts大神的作品）
    RegExr ——> 可视化正则表达式
    Regulex ——> 可视化正则表达式
    Javascript-ast Visualizer ——> 可视化抽象语法树
    compile javascript online ——> 在线编译
    GDB online Debugger ——> 在线编译
