null和undefined
Date:  2019-02-12
Tags : # JavaScript # 数据类型

一. undefined在变量声明中的体现：

1）声明了、但是没有初始化的变量：

var aa;
alert(aa);

浏览器会弹出窗口显示 undefined。因为对于使用了 var 声明但没有进行初始化定义的变量, 其值默认为 undefined.
2） 未声明的变量(1)：

alert(aa);
