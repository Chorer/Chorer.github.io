分享一些实用工具和平台

 2019-04-23   
 580 words    2 mins   


这个帖子用来不定期更新网上收集的一些实用工具和平台，运用得好还是可以提高学习或工作效率的。
PS：部分资源国内无法访问，这个问题自己想办法解决。

社区和博客 ：

StackOverflow ——> 最棒的编程问答社区
SegmentFault ——> 国内编程问答社区，得到高质量回答的机率很高
掘金 ——> 文章质量很高，社区氛围很好
Bits and pieces ——> 不错的技术博客网站（好像和Medium有关系？）
CSDN ——> 不建议去
博客园 ——> 优秀文章和大牛聚集地
FreeCodeCamp ——> FreeCodeCamp 中文社区

文件编辑类：

Online converter ——> 文件格式转换
PDF To Word ——> pdf 转 word
Online PDF to EPUB converter ——> pdf 转 epub
Table Convert Online ——> 表格转换
Compress images / Tinypng / Shrinkme——> 图片压缩
Smart Resize ——> 图片修改
Syntax-highlight-word ——> 可以在 word 中使用代码高亮
tool.lu ——> 在线编写 markdown 以及 html 和 md 转换
长链接压缩 ——> 链接太长不好看？用这个就对了
Bigjpg ——> AI 图片无损放大

电子书：

GitHub coding books online reading ——> Github编程相关书籍分享
itbook.download ——> 编程相关书籍免费下载
Jiumo Search ——> 目前用过最棒的电子书搜索网站
Ebookee ——> 编程相关书籍免费下载（外文）

搜索引擎：

chongbuluo search ——> 集成搜索网站
DuckDuckGo ——> 搜索引擎

素材：

SM.MS Picture ——> 免费图床
Font Awesome ——> 图标库
Iconfont ——> 图标网站
free icons ——> 图标网站
fontke ——> 字体网站
asciiflow ——> 在线生成字符码图
Picular ——> 根据关键词检索与之相关的色彩
Clippy — CSS clip-path maker ——> CSS3 clip 工具
Bestvist ——> CSS3 颜色渐变工具

前端相关：

MDN Web Doc ——> MDN文档
ECMAScript International ——> ECMAScript 标准
CSS速查总表 ——> 速查属性
Caniuse ——> 前端兼容性自查
compile javascript online / GDB online Debugger ——> 在线编译
Codepen ——> 代码托管和分享，可以嵌入 md 文章中
ScreenDump ——> 响应式效果测试
JavaScript Visualizer ——> 可视化JavaScript执行过程（仅ES6之前）
Loupe ——> 可视化事件循环 （Philps Roberts大神的作品）
RegExr / Regulex ——> 可视化正则表达式
Javascript-ast Visualizer ——> 可视化抽象语法树
cubic-bezier ——> 可视化贝塞尔曲线

其它

Moakt ——> 临时邮箱
